# SudokuEngine
A modular and interactive Sudoku grid engine built with HTML, CSS, and JavaScript.  
Designed for flexibility, clarity, and future scalability.

SudokuEngine provides a clean interface for generating and interacting with Sudoku puzzles of various sizes (4×4, 6×6, 9×9). It includes animated transitions, drag-and-drop number tools, and dynamic puzzle rendering. The project is structured to support future enhancements such as solving algorithms, input validation, and puzzle persistence.

🚀Features

- Multiple grid sizes: 4×4, 6×6, 9×9  
- Responsive and minimal UI  
- Drag-and-drop number tools  
- Smooth animations and transitions  
- Modular structure for easy extension

🚀Getting Started

To run the project locally:

1. Clone the repository  
2. Open `index.html` in your browser
